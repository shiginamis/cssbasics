Sta je CSS?

  • CSS je skracenica od Cascading Style Sheets
  • CSS opisuje kako ce HTML elementi biti prikazani na stranici
  • Tipicni CSS file je tekstualni file sa ekstenzijom ".css" i sadrzi komande i pravila koja govore HTML-u kako da se prikazuje.
  • Koristeci CSS, mozete kontrolisati boju teksta, stil fonta, razmak izmedju paragrafa, koje su slike i boje u pozadini koristene...
  • Prednosti CSS-a:
    1. CSS stedi vrijeme
    2. Stranice se brze ucitavaju
    3. Lagano odrzavanje
    4. Kompatibilnost sa razlicitim uredjajima
